# homebrew-kei

Homebrew tap for [kei](https://github.com/rhoopr/kei) — a photo sync engine.

## Install

```bash
brew install rhoopr/kei/kei
```

## Update

```bash
brew update
brew upgrade kei
```

## Migrating from icloudpd-rs

```bash
brew uninstall icloudpd-rs
brew untap rhoopr/icloudpd-rs
brew install rhoopr/kei/kei
```
